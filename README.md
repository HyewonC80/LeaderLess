# LeaderLess

> AI를 활용하여 ‘조장의 부담’을 줄이는 팀 프로젝트 협업 서비스

<br>

## 문제 정의

프로젝트에서는 조장에게 업무가 과도하게 집중되는 경우가 많습니다.

특히 다음과 같은 문제가 반복적으로 발생합니다.

- 역할 분배의 어려움
- 팀원의 자료 미제출 및 무임승차
- 팀원에게 독촉 메시지를 보내야 하는 부담
- 제출된 자료의 퀄리티 확인에 드는 시간
- 팀 프로젝트 진행 상황 파악의 어려움

본 프로젝트는 AI를 활용하여 조장의 관리 부담을 줄이고,
더 공정하고 효율적인 협업 환경을 만드는 것을 목표로 합니다.

---

## 핵심 기능

### AI 자료 피드백 기능
- 팀원이 제출한 자료를 AI가 분석하여
  - 자료의 부족한 점
  - 보완 방향
  - 정리 수준
등을 피드백합니다.
